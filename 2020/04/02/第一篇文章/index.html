Java | 包

包的声明

​ package 包名;

​ 作为源文件的第一条语句；

包中文件的存储

​ 源文件的存储路径为 …\包名

​ 如 源文件中使用了包语句 package tom.jiafei

​ 那么，该源文件的存储路径应为 …\tom\jiafei

包中源文件的编译

​ 对包中源文件进行编译时，要在其包的父目录运行cmd进行编译

​ 如 源文件存储路径为 D\Code\tom\jiafei

​ 那么，应在 D\Code 目录下运行cmd进行编译，编译时要给出全路径，

​ 格式为：javac 包名\主类名.java

​ 如 javac tom\jiafei\test.java

包文件的运行

​ 运行时，同样要求在父目录下运行，执行java命令时给出包名

​ 格式为：java 包名.主类名

​ 如 java tom.jiafei.test

包之间的调用

​ 调用外部的包时，要确保所调用的方法或属性是 public 级别的，否则编译时会提示权限不够
