HTML input

2020-04-08 HTML

HTML | input

  • type

    • button——按钮

    • checkbox——复选框

    • radio——单选框

    • text——文本

    • file——文件

    • email——邮箱（自动校验格式）

    • number——数字

    • range——滑动条

    • reset——重置

    • submit——提交

    • image——定义图像作为提交按钮

  • value

    value 属性规定 input 元素的值。

    value 属性对于不同 input 类型，用法也不同：

    • 对于 “button”、”reset”、”submit” 类型 - 定义按钮上的文本
    • 对于 “text”、”password”、”hidden” 类型 - 定义输入字段的初始（默认）值
    • 对于 “checkbox”、”radio”、”image” 类型 - 定义与 input 元素相关的值，当提交表单时该值会发送到表单的 action URL。
