CSS | 四种居中div的方法

  • Flex layout

    在父容器中添加以下样式

    display: flex;

    justify-content: center;

    align-items: center;

  • Margin auto

    在子容器中添加以下样式

    margin: 0 auto;

  • Text-align 和 Inline-block

    在父容器中添加以下样式

    text-align: center;

    在子容器中添加以下样式

    display: inline-block;

  • 2D Transform

    在子容器中添加以下样式

    position: absolute;

    top: 50%;

    left: 50%;

    transform: translate(-50%, -50%);
